 Airbnb Lisbon Data Analysis – Project Summary

Project Overview
This project analyzes Airbnb listings in Lisbon to identify pricing patterns, neighborhood trends, and availability distribution. The objective is to extract business insights that could help hosts optimize pricing strategies and understand market dynamics.

Key Findings:
-The average listing price in Lisbon is 284€.
-The most expensive neighborhood is Azambuja, with an average price of 1 762,40 €.
-The most common room type is Entire home/apt, representing the majority of listings.
-The neighborhoods with the highest number of listings are Santa Maria Maior.
-Availability is mainly concentrated in the Low range.

Tools Used:
Microsoft Excel (Data Cleaning, Pivot Tables, Dashboard)
Data Cleaning Techniques (Removing nulls, formatting, filtering)
Correlation Analysis
Data Visualization

Business Impact:
This analysis provides insights into market segmentation, pricing behavior, and supply distribution, helping stakeholders make data-driven decisions.
